# Roadmap

## Fase 0 — Fundação

- Documentação.
- Repositório monorepo.
- CI inicial.
- Docker Compose.
- Design system básico.
- Contratos de API.

## Fase 1 — MVP

- Autenticação.
- Eventos e cards.
- Alertas por luta.
- Push notifications.
- Painel administrativo.
- Monitoramento híbrido.
- Português e inglês.

## Fase 2 — Inteligência operacional

- Previsão dinâmica.
- Score de confiança.
- Timeline ao vivo.
- Histórico de alterações.
- Métricas de precisão.
- Widgets e Live Activities.

## Fase 3 — Expansão

- Mais organizações.
- Estatísticas avançadas.
- Assinaturas premium.
- Perfis de atletas.
- Calendário sincronizado.
- Wearables.

## Fase 4 — Plataforma

- Palpites.
- Ranking social.
- Comunidade.
- APIs para parceiros.
- Soluções B2B.
