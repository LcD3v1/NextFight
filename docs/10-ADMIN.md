# Painel Administrativo

## Stack

- React.
- TypeScript.
- Vite.
- Tailwind CSS.
- shadcn/ui.
- TanStack Query.
- React Hook Form + Zod.

## Áreas

- Dashboard geral.
- Organizações.
- Eventos.
- Cards e lutas.
- Monitor ao vivo.
- Usuários.
- Alertas e entregas.
- Assinaturas.
- Logs de auditoria.
- Saúde dos provedores.

## Monitor ao vivo

Deve permitir:

- iniciar e pausar evento;
- iniciar luta;
- atualizar round e relógio;
- finalizar luta;
- marcar no contest, cancelada ou adiada;
- reordenar card;
- disparar alerta manual;
- visualizar usuários impactados;
- desfazer uma mudança com nova entrada de auditoria.
