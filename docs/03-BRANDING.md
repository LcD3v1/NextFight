# Branding

## Marca

**Nome:** NextFight  
**Slogan principal:** Never miss the fight that matters.

## Personalidade

- Direta.
- Confiável.
- Enérgica.
- Tecnológica.
- Premium.

## Identidade visual inicial

- Base escura para destacar conteúdo ao vivo.
- Vermelho para ação, urgência e marca.
- Verde para estados ao vivo e sucesso.
- Amarelo ou laranja para atenção e previsão.
- Branco e cinza para hierarquia de texto.

## Tom de voz

- Frases curtas.
- Sem exagerar precisão.
- Informar claramente mudanças.
- Evitar jargão técnico para usuários comuns.

## Exemplos de texto

- Sua luta é a próxima.
- O card foi atualizado.
- Nova previsão disponível.
- Alerta ativado.
- Evento em pausa.
- Luta cancelada.
